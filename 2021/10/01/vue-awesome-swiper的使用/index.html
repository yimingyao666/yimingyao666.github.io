vue-awesome-swiper的使用

一、介绍

轮播是我们在项目比较的常见的效果，对于vue的项目，vue-awesome-swiper是我们经常使用的轮播插件。
但是在项目中使用vue-awesome-swiper时，经常会出现各种各样的问题（版本问题、使用方法…），
所以想写一篇记录一下，方便使用。

二、安装

npm install swiper@5 --save
npm install vue-awesome-swiper@4 --save

在此用的版本为swiper5.x版本，vue-awesome-swiper为4.x版本。

三、引入

这里使用的是局部引入的方式。

import { Swiper, SwiperSlide } from 'vue-awesome-swiper'
import 'swiper/css/swiper.css'

注意：这里要注意css的引入方式，不同的版本会有不同的路径。

引入后不要忘记注册，

components: {
    Swiper,
    SwiperSlide
}

四、使用

1.HTML部分

<swiper :options="swiperOption" ref="mySwiper" v-if="bannerList.length">
  <swiper-slide v-for="item in bannerList" :key="item.id">
    <img :src="item.img_src" alt="banner图片">
  </swiper-slide>
  <div class="swiper-pagination" slot="pagination"></div>
</swiper>

2.JS部分

data () {
    return {
        swiperOption: {
            autoplay: {
            delay: 3000,
            stopOnLastSlide: false,
            disableOnInteraction: true
            },
            loop: true,
            pagination: {
            el: '.swiper-pagination'
            }
        }
    }
},
computed: {
    swiper () {
        return this.$refs.mySwiper.$swiper
    }
}

3.修改样式

这里使用的sass穿透，可以根据样式自行设置，

.index-swiper /deep/ .swiper-pagination-bullet-active {
    background-color: #fff;
}

五、完整代码

<template>
    <div class="index-swiper">
        <swiper :options="swiperOption" ref="mySwiper" v-if="bannerList.length">
        <swiper-slide v-for="item in bannerList" :key="item.id">
            <img :src="item.img_src" alt="banner图片">
        </swiper-slide>
        <div class="swiper-pagination" slot="pagination"></div>
        </swiper>
    </div>
</template>

<script>
    import { Swiper, SwiperSlide } from 'vue-awesome-swiper'
    import 'swiper/css/swiper.css'

    export default {
        name: 'IndexSwiper',
        props: {
            bannerList: {
                type: Array
            }
        },
        components: {
            Swiper,
            SwiperSlide
        },
        data () {
            return {
                swiperOption: {
                    autoplay: {
                        delay: 3000,
                        stopOnLastSlide: false,
                        disableOnInteraction: true
                    },
                        loop: true,
                        pagination: {
                        el: '.swiper-pagination'
                    }
                }
            }
        },
        computed: {
            swiper () {
                return this.$refs.mySwiper.$swiper
            }
        }
    }
</script>

<style lang="scss" scoped>
    .index-swiper /deep/ .swiper-pagination-bullet-active {
        background-color: #fff;
    }
    .index-swiper{
        width: 100%;
        height: 545px;
        margin-top: 25px;
        background-color: #ffffff;
        border-radius: 8px;
        overflow: hidden;
        .swiper-slide img{
            width: 100%;
            height: 100%;
        }
    }
</style>

以上为在vue项目中vue-awesome-swiper的常规使用，不足之处欢迎之处。
