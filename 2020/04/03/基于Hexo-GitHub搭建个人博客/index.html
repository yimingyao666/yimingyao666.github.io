基于Hexo+GitHub搭建个人博客

一、环境准备

1、安装git：https://git-scm.com/downloads

根据自己的系统选择下载安装

2、安装Node：https://nodejs.org/en/

选择稳定版进行下载安装

3、检查是否安装成功

windows: win+R
mac: 打开终端
检查git：输入git --version
检查node：输入node -v

看到版本号即为安装成功

4、npm是node提供的包管理器

为了方便快速的下载所需的应用包，建议安装一下淘宝的镜像：npm install -g cnpm --registry=https://registry.npm.taobao.org

检查是否安装成功即：cnpm -v

二、安装Hexo博客框架

1、使用命令：cnpm install -g hexo-cli

待安装完后输入：hexo -v，检查是否安装成功

三、搭建自己的博客

1、建一个文件夹，起名为blog或myblog等，名字自己来定义

ps：记好自己所建文件夹的位置，如果后续有什么问题，将文件夹删掉，重新操作即可

2、进入自己所建文件夹中，以blog为例：cd blog

3、初始化博客：hexo init

这个过程需要一点时间，请耐心等待一会儿。
当看到Start blogging with Hexo!时即为成功。

4、启动博客：hexo s

5、停止启动：control c

然后在浏览器输入：http://localhost:4000，即可看到自己的博客。

四、写自己的第一篇博客

1、创建文章：hexo n “题目”，例如：hexo n “我的第一篇博客文章”

你可以在source/_post文件夹中看到这篇文章，默认是Markdown格式，你可以使用vs code或其他编辑器打开，创作自己的文章。

ps：如果使用vs code建议安装一下扩展：Markdown Preview Enhanced、Markdown Preview Github Styling，这样可以在编辑器中看到写成的样式。

2、查看文章

·确定在blog位置中
·hexo clean：清理缓存
·hexo g：生成静态文件
·hexo s：启动博客

五、将自己的博客部署到GitHub上面

1、进入到自己的GitHub中

2、新建一个仓库

输入名称、描述即可

ps：命名必须符合：自己的昵称 + .github.io

3、安装一个Git上传的插件：cnpm install --save hexo-deployer-git

4、在_config.yml中配置Deployment

repo为自己的仓库名

5、部署到GitHub中：hexo d

GitHub中会有上传的文件

6、在浏览器中输入你的github仓库名即可看到自己的博客

六、更换博客主题（以yilia主题为例）

1、下载主题：git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

在文件夹themes下会有yilia文件夹

2、配置新主题

3、hexo其他主题地址：https://hexo.io/themes/

4、本地查看：hexo s

5、部署到github中

·确定在blog位置中
·hexo clean：清理缓存
·hexo g：生成静态文件
·hexo d：部署博客

到这里，从环境搭建到部署已经完成，之后可以对自己的博客进行进一步优化。
